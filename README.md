### 6. Creating Seasonal Categories

```python
def get_season(month):
    if month in [12,1,2]: return "Winter"
    elif month in [3,4,5]: return "Summer"
    elif month in [6,7,8]: return "Monsoon"
    else: return "Autumn"
df['Season'] = df['Month'].apply(get_season)
```
